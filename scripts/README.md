# Utility Scripts

This directory contains utility scripts for development and maintenance.

## Available Scripts

### `setup-env.sh`
Sets up environment variables for local development.

```bash
./scripts/setup-env.sh
```

### `verify-env.sh`
Verifies that all required environment variables are set correctly.

```bash
./scripts/verify-env.sh
```

### `diagnose-backend.sh`
Diagnoses backend container issues and provides troubleshooting information.

```bash
./scripts/diagnose-backend.sh
```

### `show-backend-error.sh`
Shows recent backend error logs from Docker container.

```bash
./scripts/show-backend-error.sh
```

## Archived Scripts

Old deployment and troubleshooting scripts have been moved to `archive/` for reference.
These are no longer actively maintained but kept for historical purposes.
